Project này có gồm 2 phần:

+ Ứng dụng gọi món, quảng cáo, quản lý thông tin cho khách hàng xây đựng trên nền tảng ReactJS cho mobile. Sở hữu bởi Tạ Quốc Thảo (https://github.com/taquocthao/delivery-food-app)

+ Web API (.Net framework) lưu trữ database, thực hiện xử lý request từ ứng dụng client. Đồng thời chứa trang admin cho phép quản lý các yêu cầu order gửi về và quản lý cơ sở dữ liệu chính

- Repository này chỉ quản lý Web API
- Demo: http://ldvmservices.somee.com/
